## Web responsiva

O #Sustenido funciona pelo navegador no computador, tablet e celular, inclusive iPhone. O manual principal descreve essa experiência web responsiva.

## Aplicativo React Native

O aplicativo nativo possui biblioteca, edição, calendário, recursos sociais e suporte offline em evolução. Ele ainda não tem paridade com toda a aplicação web.

## Diferenças importantes

| Recurso | Web responsiva | Aplicativo nativo |
|---|---|---|
| Biblioteca e músicas | Entregue | Implementado em partes |
| Calendário e social | Entregue | Implementado em partes |
| Offline | Beta | Beta, implementação própria |
| Afinador | Entregue na web | Planejado |
| Apresentação completa | Superfície principal | Sem paridade confirmada |

Use a web quando precisar do conjunto completo de apresentação e ferramentas. Não presuma que uma instrução da web se aplica à mesma tela no aplicativo.
